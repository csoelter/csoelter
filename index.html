    Die Projekte sind im Rahmen des freeCodeCamp curriculums zur Erlangung der Zertifikate enstanden (klicken für eigene Seite).

    Pomodoro-Clock

    Die Pomodoro-Technik ist eine Methode zum Zeitmanagement. Die Arbeit wird dabei in Abschnitte von 25 Minuten Arbeitszeit und kurzen Pausen (5 Minuten) eingeteilt. Die Pomodoro-Uhr wurde in Javascript (React) geschrieben und wechselt automatisch nach 25 Minuten in den 5 minütigen Pausenmodus. Dannach startet der die Uhr wieder mit einer 25 Miunuten Arbeitsphase. Arbeitsphasen und Pausen sind anpassbar. Der Quellcode zur Pomodoro-Clock kann hier eingesehen werde.

    Taschenrechner

    Einfacher Taschenrechner, der die Grundrechenarten beherrscht. Geschrieben im Javascript (React). Der Quellcode zum Taschenrechner kann hier eingesehen werde.

    Drum-Maschine

    Einfache Drum-Maschine, die sowhl über die Tastatur, als auch über die Buttons steuerbar ist. Geschrieben in JavaScript (React). Der Quellcode zur Drum-Maschine kann hier eingesehen werde.

    Markdown-Previewer

    Markdown ist eine vereinfachte Auszeichnungssprache. Sie verwendet in der Rohform vor allem Satzzeichen um die eigentliche Darstellung zu beschreiben. Markdown wird überwiegend von technikaffinen Seiten wie GitHub oder StackOverflow verwendet. Der vorliegende Editor wandelt Markdown-Code in XHTML um und wurde in JavaScript (React) geschrieben. Der Quellcode zum Markdown-Previewer kann hier eingesehen werde.
